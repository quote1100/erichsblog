This is a Rock Paper Scissors game I built using Java. The goal of the project was to simulate a game where the computer has a higher chance of winning—basically giving the bot a bit of an edge. It was one of the first programs I ever created, so while it's simple, it gave me a chance to practice using Java, Git, and general programming concepts.

There are two bot difficulty levels:

  • Easy: Makes completely random choices.
  • Hard bot: Uses a bit of strategy:
    • On the first round, it chooses randomly.
    • After that:
      • If it won the last round, it tends to copy your previous move.
      • If it lost, it's more likely to choose the move that beats what you played last time—anticipating you might repeat it.
    • There’s always some randomness to avoid being too predictable.
